ADMIN

1. Overview

As an Admin, you have full access to every module in the system. You are responsible for setting up the application, managing users, configuring products, and overseeing the entire manufacturing lifecycle from order creation to final dispatch.

Tip: Bookmark the Dashboard page — it gives you a real-time snapshot of all production activity.

2. Getting Started

Login

Open the app URL in your browser. Enter your email and password, then click Sign in.

Your session lasts 7 days. After that, you'll need to log in again.

4. Orders

The Orders module is where manufacturing begins. You can view, create, and manage all production orders.

Order Status Flow

CREATED→
ACTIVE→
IN PROGRESS→
COMPLETED

Creating an Order

Steps:

  • Go to Orders tab
  • Click + New Order
  • Select the Product, Client, and enter Quantity
  • Set Voltage, Motor Type, and Due Date
  • Click Create — the system auto-generates serial numbers and barcodes for each unit

Order Detail Page

Click any order to see its full details:

  • Summary card — order number, client, product, quantity, due date, progress %
  • Stage breakdown — units grouped by manufacturing stage with status badges
  • Barcodes — barcode labels for each stage (PS, BB, AY, QC, FA)
  • Dispatch status — which units are ready for shipping

6. Admin Panel

The Admin tab gives you access to all system configuration:

Users

  • Create and manage user accounts
  • Assign roles (Admin, PM, Employee, Sales, etc.)
  • Enable/disable accounts
  • Enroll face verification for biometric login

Products

  • Manage product catalogue
  • Set product codes (used in barcodes)
  • Activate/deactivate products

Clients

  • Manage customer database
  • Set billing/shipping addresses
  • Configure GST numbers
  • Mark as Global or Indian (affects tax)

BOM (Bill of Materials)

  • Define materials needed per product per stage
  • Link raw materials to production stages
  • Set quantities per unit

Checklists

  • Configure AI quality checks per stage
  • Define what photos are required
  • Set pass/fail criteria

Vendors

  • Manage supplier database
  • Set vendor categories
  • Enable vendor portal access

Box Sizes

  • Define packing box dimensions
  • Set box weights
  • Used by packing team during dispatch

Settings

  • Company details (name, address, GSTIN)
  • LUT number for exports
  • Bank account for invoices
  • Price breakdown factors

8. Inventory Module

Track all raw materials, stock levels, and movements.

  • Materials — view all raw materials with current stock levels
  • Low Stock Alerts — materials below minimum stock highlighted in red
  • Stock Movements — track every IN/OUT with batch codes
  • GRN — goods receipt from purchase orders
  • Job Cards — material dispatch to production employees

9. Approvals

As Admin, you can approve work at any manufacturing stage. Go to Approvals to see all units in WAITING_APPROVAL status. Click a unit to review and approve or reject the work.
