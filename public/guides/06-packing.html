1. Overview

As a Packing team member, your job is to pack completed controller units into boxes for dispatch. You create dispatch orders, scan units into boxes, seal boxes with photos, and submit the packed DO for approval.

4. Complete Packing Workflow

This is your primary daily workflow, step by step:

Step 1: Find Ready Orders

Go to the Packing tab. The Ready section shows orders with units that have completed Final Assembly and are ready for dispatch.

Step 2: Create a Dispatch Order (DO)

  1. Click on a ready order
  2. Click Create Dispatch Order
  3. A DO number is auto-generated (format: TSM/DO/YY-YY/NNN)
  4. Select how many units to dispatch (can be partial — not all units need to ship together)

Step 3: Set Box Count

  1. Open the DO and click Set Boxes
  2. Enter the number of boxes needed
  3. Select a box size for each box (predefined by Admin)
  4. Box labels are auto-generated: {DO Number}-BOX-1of3, BOX-2of3, etc.

Step 4: Scan Units into Boxes

  1. Open a box to start scanning
  2. Scan the unit's barcode using the barcode scanner
  3. The system validates: Does the unit belong to this order? Has it completed FA? Is it already packed?
  4. If valid, the unit is added to the box
  5. Repeat for all units in this box
Important: You can only scan units that belong to the correct order, have completed Final Assembly, and haven't been packed already. The system will reject invalid scans.

Step 5: Seal Boxes

  1. When a box is full, click Seal Box
  2. Take a photo of the sealed box (required)
  3. Upload the photo
  4. The box is now sealed — you cannot add or remove units after sealing

Step 6: Submit for Approval

  1. Once all boxes are sealed, the Submit button becomes active
  2. Click Submit — this generates a Packing List
  3. The DO moves to SUBMITTED status
  4. Accounts team will review and approve

Tip: You can print box labels from the DO detail page. Each label shows the DO number, box number, and box dimensions.

5. If a DO is Rejected

If Accounts rejects a DO, it goes back to PACKING status. You'll need to:

  • Check the rejection reason from Accounts
  • Fix the issue (may need to unseal and rescan)
  • Re-submit for approval
