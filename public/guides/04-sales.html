4. Creating a Proforma Invoice (PI)

This is your primary workflow — creating a PI that will eventually become a production order.

  1. Go to Invoices tab and click + New Proforma
  2. Select the Client (or create a new one from the Clients tab first)
  3. Choose Invoice Type: Global (export) or Indian (domestic)
  4. Set Currency: USD for global/export clients, INR for Indian/domestic clients. If USD, set the USD-INR exchange rate in the proforma.
  5. Add line items: select product, enter quantity, unit price, voltage range, HSN code
  6. Set Payment Terms, Delivery Days, and Terms of Delivery
  7. Optionally enable Split Invoice (separates goods and service charges)
  8. Click Save as Draft to save, or Submit for Approval to send to Accounts

Proforma Invoice Flow

DRAFT→

Tip: You can only edit a PI while it's in DRAFT status. Once submitted for approval, it's locked.

5. Invoice Tab (Tax Invoices)

After a dispatch order is approved by Accounts, tax invoices are auto-generated. You can view them here.

Sub-tabs

Current (This Month)

  • Shows this month's invoices
  • Grouped by Dispatch Order
  • All folders open by default

History (Past Months)

  • All past invoices grouped by month
  • Months collapsed by default
  • Search bar to find specific invoices

Each invoice row shows: invoice number, amount, item count, and tracking status.

You can only view invoices from your own proformas. You cannot see other sales team members' invoices.

Tracking Information

Once the shipping team adds a tracking number, you'll see a tracking badge on the invoice. This is read-only for you — only Accounts/Admin can edit tracking.

6. Status Tab (Order Tracking)

Track how your orders are progressing through production:

Sub-tabs

Processing

  • Orders still in production
  • Shows: phase label, progress bar, stage badges
  • Due date and hold reason (if any)
  • Click card to see full order detail

Completed

  • All fully dispatched orders
  • Dispatch summary with dates

Order Detail Page (Sales View)

When you click an order from the Status tab, you see a simplified view:

  • Summary — order number, status, ETA, progress bar, stage badges
  • Hold reason — if PM has paused production, the reason appears here
  • Invoices section — each approved dispatch order with DO number, unit count, tracking info, and download buttons for tax invoices
  • Notes — message thread with PM. Use this to ask about delays, request ETA updates, or communicate requirements

7. Clients Tab

Manage your client database:

  • View all clients with their code (auto-generated: CLI001, CLI002...)
  • Add new clients with: customer name, billing address, shipping address, GST number
  • Mark clients as Global or Indian (affects GST calculation on invoices)
  • Activate or deactivate clients
